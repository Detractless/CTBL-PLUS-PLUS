# CTBL++

A community-built add-on for Cold Turkey Blocker. Requires the paid version - this is not a crack or bypass.

Cold Turkey is one of the best blocking tools out there, but there are features the developer has said they don't plan to build. CTBL++ fills that gap. It runs alongside Cold Turkey and extends it with new lock types, enforcement mechanics, and features the community actually wants.

## What's built so far

**Queued Delay** - every unblock request, exception, or setting change gets held in a signed queue for a mandatory cooldown period before it executes. You can't cancel it, backdate it, or impulse-click your way around it. The delay is enforced at the service level, not the UI level.

## What's coming

A local AI feature that automatically categorizes sites, searches, and apps against your stated goals - and adds them to the right blocklist without you having to manage it manually.

## Who this is for

People who feel like Cold Turkey's current options aren't quite enough. If you've ever wished you could add friction to your own unblocking, set stricter rules than the app currently allows, or just want more control over how your blocks behave - this is that project.

## Current status

The core architecture is in place and the project is currently transitioning away from a standalone desktop UI toward patching Cold Turkey's own interface directly. Alongside that, the focus is on fixing known bugs, removing dead code, finishing off incomplete parts of the core feature set, and simplifying anything that got overly complex along the way.


## Get involved
